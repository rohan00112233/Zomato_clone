# Zomato Clone (Django)

## Overview
A full-stack food delivery web application inspired by Zomato.

## Features
- Custom User Model (Phone + Address)
- User Signup System
- Admin Panel for User Management

## Tech Stack
- Django
- HTML, CSS, Bootstrap
- SQLite

## How to Run

```bash
git clone https://github.com/rohan00112233/Zomato_clone.git
cd Zomato_clone
python -m venv venv
venv\Scripts\activate
pip install django
python manage.py migrate
python manage.py runserver
